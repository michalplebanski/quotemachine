AUTHOR: Michał Plebañski

TITTLE: Random quote machine

TECHNOLOGIES USED: html5, css3, jQuery, ajax

DESCRIPTION: This project uses Ajax technology. By clicking on the 'Random quote' button, the machine will show
a random quote from the url. We take a quote using 'getQuote()' in script.js file.
We can also create a Tweet by clicking the button under 'Random quote'. It has a
character limit (140)
